# Sanitization Notes

This public repository is a sanitized case study based on real commercial automation work.

The original production project remains private under client confidentiality. Platform names such as Guesty, Hostaway and HubSpot are kept because they describe the technical integration context.

All client-specific data and secrets were removed or replaced. This public version intentionally replaces private implementation details with generic fields, sanitized sample data and representative integration patterns.

## Removed

- API keys
- access tokens
- `.env` files
- production logs
- real client names
- real customer data
- real property names
- real company names
- real HubSpot private property names
- private deployment details

## Replaced with

- sanitized sample data
- generic field names
- fake HubSpot client
- public sync runner
- generic PMS records

## Safety boundaries

- No original git history is included.
- No private repository metadata is included.
- No real network calls are made.
- No production credentials are required.
- The repository does not claim to be production source code.
